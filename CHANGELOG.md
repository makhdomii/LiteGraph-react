# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Initial release of React-compatible LiteGraph package
- `GraphCanvas` component for rendering graph editor
- `useGraph` hook for programmatic graph manipulation
- `GraphProvider` and `GraphContext` for state management
- Full TypeScript support with type definitions
- Example components demonstrating usage
- Vite build configuration
- Comprehensive README with usage examples

### Features
- React component wrapper for LiteGraph.js
- Hooks API for graph manipulation
- Context API for state management
- TypeScript definitions
- Theme support (basic)
- Event callbacks (onReady, onChange, onNodeSelected, etc.)
- Support for all LiteGraph features
- Programmatic node creation and manipulation
- Graph serialization/deserialization
